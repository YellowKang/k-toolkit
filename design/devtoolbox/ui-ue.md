# UI/UE 规范与优化

## 1. 设计语言

### 色彩体系（Dark 默认）

```
主背景     --bg:           #09090f   深黑，轻微蓝紫调
次背景     --bg2:          #0d0d1a
强调色     --accent:       #8b5cf6   紫色主调
强调色2    --accent2:      #7c3aed
强调色3    --accent3:      #c4b5fd   淡紫（高亮文字）
霓虹蓝     --neon:         #06b6d4
危险色     --accent-pink:  #f43f5e
正文       --text:         #f1f5f9
次要文本   --text-muted:   rgba(241,245,249,0.65)
毛玻璃     --glass:        rgba(255,255,255,0.04)
毛玻璃边框 --glass-border: rgba(255,255,255,0.08)
```

每个工具卡片有独立 `--card-color`，用于图标背景和 hover 光晕。

### 圆角与阴影

```
--radius: 18px
--shadow-card: 0 8px 32px rgba(0,0,0,0.4), 0 1px 0 rgba(255,255,255,0.05) inset
```

### 动效曲线

```
--ease-spring: cubic-bezier(0.34,1.56,0.64,1)  弹性（卡片 hover、收藏按钮）
--ease-out:    cubic-bezier(0.16,1,0.3,1)       顺滑退出（页面切换、侧边栏）
```

### 字体

```
正文：'Inter', 'SF Pro Display', -apple-system, sans-serif
代码/结果区域：monospace
```

背景网格：48×48px 点阵，rgba(102,126,234,0.04)，增强科技感而不干扰内容。

## 2. 布局结构

```
┌────────────────────────────────────────────────┐
│  Topbar（64px）  ☰  搜索框  主题按钮 ⌘K提示    │
├──────────┬─────────────────────────────────────┤
│ Sidebar  │  Content 区（overflow-y: auto）      │
│ 256px    │                                     │
│ 可折叠   │  首页：收藏区 + 最近区 + 分类工具区   │
│ 至 64px  │  工具页：tool-header + #toolBody     │
│          │  搜索页：搜索结果卡片列表            │
│ Logo     │                                     │
│ 分类导航 │                         [↑回到顶部] │
└──────────┴─────────────────────────────────────┘
移动端：侧边栏 off-canvas（translateX），overlay 遮罩
移动端底部：bottom-nav（首页 / 搜索 / 收藏 / 最近）
```

### 响应式断点

| 断点 | 布局变化 |
|------|----------|
| > 768px | 侧边栏常驻，可折叠至 64px（图标模式）|
| ≤ 768px | 侧边栏 off-canvas，bottom-nav 显示 |

## 3. 组件规范

### 工具卡片

```
┌──────────────────────────────┐
│  [图标]  工具名称    [★收藏] │
│          工具描述（2行截断） │
│          [分类标签] [NEW]    │
└──────────────────────────────┘

hover 效果：
  - translateY(-4px) + 放大阴影
  - 图标背景光晕（--card-color 半透明）
  - ease-spring 弹性动画

收藏按钮：
  - 右上角 ★，悬浮才显示（已收藏常显）
  - 点击就地切换，不触发页面重渲染

NEW 标签：
  - isNew: true 时展示，accent 色背景

用量 badge：
  - usageCounts[id] > 0 时展示使用次数
```

### 按钮规范

| 类型 | 用途 | 样式 |
|------|------|------|
| `.btn-primary` | 主操作（格式化、转换、生成）| accent 紫色背景 |
| `.btn-secondary` | 次要操作（清空、复制、下载）| glass 半透明背景 |
| `.btn-danger` | 危险操作 | accent-pink 背景 |

### 搜索框

```
- 占 topbar 中间区域，最大宽度 480px
- 左侧搜索图标，右侧清空按钮（有内容时显示）
- 聚焦时边框高亮（accent 色）
- 快捷键提示：placeholder 中注明 / 或 ⌘K
```

### 命令面板

```
- 全屏遮罩 + 居中弹层（max-width: 600px）
- 顶部搜索输入，下方结果列表
- 选中项高亮背景（glass）
- 每条目：图标 + 工具名 + 分类标签
- 键盘：↑↓ 导航，Enter 确认，ESC 关闭
```

### Toast 提示

```
- 底部居中浮层，z-index 最高
- 自动消失（约 2s）
- 文案简短：「已复制」「已收藏」「链接已复制」
```

### 顶部进度条

```
- id="top-progress"，工具懒加载时显示
- accent 色线条，从左到右动画
- 加载完成后 hideProgress() 隐藏
```

## 4. 页面状态规范

### 首页空状态

| 页面 | 空状态文案 |
|------|------------|
| 收藏页（无收藏）| 「暂无收藏」+ 「点击工具卡片右上角 ★ 收藏常用工具」|
| 最近使用（无记录）| 「暂无最近使用」+ 引导打开工具 |
| 搜索无结果 | 「未找到「xxx」」+ 「试试其他关键词」|

### 工具加载状态

```
加载中：骨架屏占位（灰色矩形脉冲动画）
加载成功：工具内容区直接替换骨架屏
加载失败：重试按钮 + 错误提示
```

### 反馈时机

| 操作 | 反馈 |
|------|------|
| 收藏 / 取消收藏 | Toast + 按钮就地切换 |
| 复制成功 | Toast「已复制到剪贴板」|
| 分享链接 | Toast「链接已复制」|
| 主题切换 | 图标即时变化，无闪烁 |
| 工具加载 | 顶部进度条 |

## 5. 已知问题与优化建议

| 优先级 | 问题 | 建议 |
|--------|------|------|
| 高 | 移动端工具详情页 header 内容过多，易折行 | 分享按钮改为 icon-only 或收入 kebab menu |
| 高 | SW 更新后无用户感知提示 | 消费 sw-updated 消息，显示「发现新版本，点击刷新」Toast |
| 中 | 搜索下拉无数量上限，结果过多时滚动体验差 | 单分类最多 3 条，底部加「查看全部 N 个结果」|
| 中 | 工具页无同分类推荐 | 工具详情底部加同分类推荐卡片（3个）|
| 中 | 侧边栏折叠后分类文字消失，无 tooltip | PC 折叠态为每个分类项加 title tooltip |
| 低 | dark 模式部分亮色 --card-color 对比度不足 | 统一在 --card-color 基础上强制最低亮度阈值 |
| 低 | 回到顶部 FAB 固定位置可能遮挡工具操作区 | 工具页隐藏 FAB，或增加 bottom offset |
